# Codebase Audit Skill
1. Use sub-agents to audit in parallel: security, code quality, test coverage, standards compliance
2. Categorize findings as Critical/High/Medium/Low
3. Cross-reference against CLAUDE.md coding standards
4. Before finalizing, verify each finding against actual code (no false positives)
5. Output a numbered remediation plan with effort estimates
6. Exclude any work already completed in the codebase
